Getting Started

Download ShoppingStyle.js from pub/javascripts. At the every beginning, you need to add those script on your html.

<script src="https://ajax.googleapis.com/ajax/libs/jquery/3.5.1/jquery.min.js"></script>
<script type="text/javascript" src="<folder containning ShoppingStyle.js>/ShoppingStyle.js"></script>

Then you are able to use ShoppingStyle Library!

Constructors:

ImageContainer()

Initialize an empty image container which user can add more than one image to it.
The very first image will displayed on webpage.

const imageContainer = new ImageContainer()

LayoutGenerator()

Initialize an empty list of images which we will create layout on.

ImageContainer Methods

.addFirstPicture(image, selector, style)

add image to selected node with style. Then the image will appended as child element of selected element.

image: the link to the image; type:string

selector: the parent node we want to select and add image to; type:string

style: the html inline style of the image; type:string

imageContainer.addFirstPicture("image.png", "#parent_id", "width: 750px; height: 350px; margin: 10px;")

.addNextPicture(image)

append image to the container. Only call this function if .addFirstPicture() is already called.

image: the link to the image; type:string

imageContainer.addNextPicture("image.png")

.loopPictureHelp(delay)

loop images in current container one time, each image will show on webpage one by one.

delay: the timeout between each image change; type: number

imageContainer.loopPictureHelp(2000)

.loopPictureAuto(delay)

loop images in current container again and again with no stopping, each image will show on webpage one by one.

delay: the timeout between each image change; type: number

imageContainer.loopPictureAuto(2000)

.findNextPict(selector)

change current image(which displayed on the webpage) to the next image in container.

selector: the id/classname of the button to be selected, if the button clicked, the method will be triggered; type: string

imageContainer.findNextPict("#button_id")

.findPrePict(selector)

change current image(which displayed on the webpage) to the previous image in container.

selector: the id/classname of the button to be selected, if the button clicked, the method will be triggered; type: string

imageContainer.findPrePict("#button_id")

.changeByOne(selector)

change current image(which displayed on the webpage) to the second image in container
when mouse is over current image, and change back to current image if mouse leaves image.

selector: the id/classname of the parent node; type: string

imageContainer.changeByOne("#pic_container")

LayoutGenerator Methods

.addObjects(object, selector, style)

add images which we want to put in the layout.

object: the link to the image; type:string

selector: the parent node we want to select and add image to,
only one LayoutGenerator for one parent node.
Cannot add elements to parent node other than current LayoutGenerator; type:string

style: the html inline style of the image; type:string

layout.addObjects("image.png", "#parent_id", "width: 750px; height: 350px; margin: 10px;")

.increasing(selector)

arrange pictures in increasing order by width.

selector: the parent node we want to select and modify the layout; type:string

layout.increasing("#parent_id")

.decreasing(selector)

arrange pictures in decreasing order by width.

selector: the parent node we want to select and modify the layout; type:string

layout.decreasing("#parent_id")

.alignRight(selector)

align pictures to parent's right border .

selector: the parent node we want to select and modify the layout; type:string

layout.alignRight("#parent_id")

.alignLeft(selector)

align pictures to parent's left border .

selector: the parent node we want to select and modify the layout; type:string

layout.alignLeft("#parent_id")

.resize(percentage, selector)

resize all pictures by percentage of their current size.

selector: the parent node we want to select and modify the layout; type:string

layout.resize("#parent_id")
